# ATM-Simulation

**Project Overview**

The ATM Simulation System is a Python-based console application that replicates fundamental banking operations in a structured and interactive environment.

This project simulates core ATM functionalities such as secure authentication, balance management, deposit and withdrawal processing, and transaction tracking. It demonstrates strong foundational programming skills and real-world logic implementation.

**Key Features**

Secure PIN-based authentication

Balance inquiry functionality

Deposit operation with dynamic balance update

Withdrawal operation with insufficient balance validation

Transaction history tracking

Menu-driven user interface

**Tech Stack**

**Programming Language:** Python

**Execution Environment:** Jupyter Notebook / VS Code

**Concepts Applied:**

Functions

Loops

Conditional Statements

Exception Handling

Lists (for transaction tracking) Exception handling for invalid inputs

**System Workflow**

User enters a secure PIN for authentication

Upon successful login, the system displays transaction options

Selected operations update the account balance dynamically

All transactions are recorded for reference

The session ends securely upon exit
